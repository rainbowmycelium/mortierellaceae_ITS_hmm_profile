# Mortierellaceae ITS hmm profile

Fasta files and hmm profile enabling the search of ITS region in _Mortierellaceae_ fungi. 

The hmm profile available here was created based on a subset of ITS (Internal Transcribed Spacer) sequences of _Mortierellaceae_ fungi available in GenBank NCBI database. The sequences were aligned with MAFFT v7.526 (https://mafft.cbrc.jp/alignment/software/). Hmm profile was created from the alignment using HMMER 3.3.2 (http://hmmer.org/).


# Files

_ITS_mortie_GB_for_hmm.fasta_ - fasta file with raw ITS sequences downloaded from GenBank NCBI

_ITS_mortie_GB_hmm_al.fasta_ - fasta file with aligned ITS sequences downloaded from GenBank NCBI

_ITS_mortie_GB.hmm_ - hmm profile created based on the sequences
