# mortierellaceae_ITS_hmm_profile
Fasta files and hmm profile enabling the search of ITS region of Mortierellaceae fungi. Financing information: Projekt finansowany ze środków budżetu państwa, przyznanych przez Ministra Nauki w ramach Programu „Perły nauki II” ("Interakcje glonów i pierwotnych grzybów lądowych", PN/02/0042/2023)
